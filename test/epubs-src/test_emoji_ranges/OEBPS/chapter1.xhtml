Emoji Range Test

This EPUB mirrors the current extra Unicode intervals from the font-generation script.

Use it after regenerating fonts so you can quickly see whether coverage changed.

Chapter 2 covers the kept music symbols and accidentals, hand gestures, and the heart symbol.

Chapter 3 covers the full emoticons block: U+1F600 through U+1F64F.

Chapter 4 covers the exact non-emoji fallback slice: lambda, the targeted Cyrillic subset, and script small l.

Every sample is labeled with its Unicode code point so missing glyphs are easy to trace back to the interval list.
