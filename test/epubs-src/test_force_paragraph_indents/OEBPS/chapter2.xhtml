Should Gain Indents

PASS when Force Paragraph Indents is ON: both paragraphs below should start with a first-line indent.

PASS when Force Paragraph Indents is OFF: both paragraphs below should stay flush left.

Case A: Explicit text-indent: 0

This paragraph explicitly asks for no indent. The force-indent setting should override that only when the feature is turned on, making the first line step in from the left edge.

A second paragraph makes the difference easier to spot because the start of each block should line up differently when you toggle the setting.

Case B: Explicit text-indent: 0 with justified text

This paragraph is justified and still declares text-indent zero. The force-indent setting should add the same first-line indent here instead of leaving it as a solid wall of text.

If the feature is working, the first line starts indented but the rest of the paragraph still uses normal justified layout.
