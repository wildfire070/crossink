Alignment

Indent Variants

Zero indent plus justify: This paragraph declares text-indent zero and justified alignment. Force Paragraph Indents should be able to add an indent when that setting is enabled.

Positive indent: The first line should start inward, while following lines return to the normal left edge. Extra Paragraph Spacing may suppress this unless Force Paragraph Indents is enabled.

Hanging indent: The first line starts farther left than the following lines, like a definition list or bibliography entry. This checks negative text-indent handling.

Margins: This justified paragraph has left and right margins. Text should stay inside the narrower area, and guide dots should be placed between words inside that area.

Nested block: This paragraph lives inside an indented container. It should stay shifted right without pushing text off-screen.
